Assignment 8 Sorting Algo
------------------------------------------------------------------------------------------------------------------------
This folder contains the following files; main.cpp, Sort.cpp, Sort.h that work in Atom and Clion using c++11.
------------------------------------------------------------------------------------------------------------------------
NOTE: Run Docker first, then use terminal and attach compsci.
To run program, open terminal or command line and type in the following: g++ -std=c++11 *.cpp -o main
Once you have run the above g++ command, run as and example; ./main 100k.txt
100k.txt is the name of the file you want to sort

Once ran, you will get an output to the screen showing you the start time, end time and elapsed time of the Sorter.
